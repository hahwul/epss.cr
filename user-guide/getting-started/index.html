Getting Started

Prerequisites

Requirement Version
Crystal >= 1.20.2

epss.cr is pure Crystal with no native dependencies — it runs anywhere Crystal does. The only external services it touches are the public FIRST EPSS API and the public daily CSV feed.

Installation

Add the dependency to your shard.yml:

dependencies:
  epss:
    github: hahwul/epss.cr

Then install:

shards install

Your First Program

Create hello.cr:

require "epss"

if score = EPSS.score("CVE-2022-27225")
  puts "CVE        : #{score.cve}"
  puts "EPSS       : #{score.epss}"
  puts "Percentile : #{score.percentile}"
  puts "Date       : #{score.date.try(&.to_s("%Y-%m-%d"))}"
  puts "Band       : #{score.band}"
end

Run it:

crystal run hello.cr

EPSS.score(cve) returns EPSS::Score? — nil when FIRST has no published score for the CVE.

Batch lookup

EPSS.scores(cves) batches into chunks of 100 CVEs per request and returns a flat Array(EPSS::Score):

EPSS.scores(["CVE-2024-3094", "CVE-2021-44228", "CVE-2014-0160"]).each do |s|
  puts "#{s.cve}\t#{s.epss}\t#{s.band}"
end

Non-raising parse

For JSON input arriving from another system, use EPSS.from_json? to get nil on malformed payloads instead of an exception:

if scores = EPSS.from_json?(payload)
  scores.each { |s| store(s) }
else
  log_invalid_payload(payload)
end
